# IntelHQ

IntelHQ is a real-time intelligence platform for monitoring geopolitical,
technology, and financial developments in one place.

This repository is now maintained as IntelHQ, with product direction,
architecture, and feature scope evolving beyond the original fork.

## What IntelHQ delivers

- Unified global intelligence dashboard with live map-based context
- Variant-focused experiences for world, technology, and finance tracking
- AI-assisted summaries and signal aggregation across multiple sources
- Web and desktop delivery from one shared codebase

## Product surfaces

- Web: <https://intelhq.io>
- Tech: <https://tech.intelhq.io>
- Finance: <https://finance.intelhq.io>

## Development

```bash
npm install
npm run dev
```

Useful commands:

- `npm run build` - Production web build
- `npm run typecheck` - TypeScript checks
- `npm run test:data` - Data/unit-style test suite
- `npm run test:e2e:runtime` - Runtime Playwright coverage
- `npm run desktop:dev` - Tauri desktop development
